# Tria

Runna, but for Triathlons.

## Mobile Frontend

The mobile application is built with React Native and Expo.

### Prerequisites

- Node.js
- npm or yarn

### Getting Started

1. Navigate to the mobile directory:
   ```bash
   cd mobile
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npx expo start
   ```

   This will start the Metro bundler. You can open the app in:
   - [Expo Go](https://expo.dev/go) by scanning the QR code
   - iOS Simulator (press `i`)
   - Android Emulator (press `a`)
